### 3. 文件的校验值计算

### 校验算法类型：
| 校验算法类型 | 校验值长度 | 校验值类型           | 安全级别 | 计算效率 | 应用场景                                     |
| ------------ | ---------- | -------------------- | -------- | -------- | -------------------------------------------- |
| CRC(32/64)   | 4/8个字节  | 常称为校验码         | 最弱     | 最高     | 传输数据的校验，客户端和文件端要传输一个文件 |
| MD5          | 16个字节   | 常称为hash值，散列值 | 居中     | 居中     | 文件的校验和数据的签名                       |
| SHA1         | 20个字节   | 常称为hash值，散列值 | 最高     | 最低     | 文件的校验，文件的唯一标识                   |

如果对安全性要求更高则可以使用sha256
